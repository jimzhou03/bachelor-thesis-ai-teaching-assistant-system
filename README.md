## Purpose of this Repository
This repository provides the current draft and summary of my bachelor thesis for application to the M.Sc. Artificial Intelligence Engineering programme at the University of Passau.

## Status
- Current version: Draft (in progress)
- Expected completion: June 2026

## Abstract (Short)
This thesis focuses on designing and implementing an AI teaching assistant system for university-level Machine Learning courses. The system integrates retrieval-augmented generation (RAG) with a domain-specific knowledge graph to provide reliable question answering, personalized exercise generation, and learning path recommendation.

## Current Progress

- Knowledge graph schema and database implementation completed
- RAG-based QA pipeline implemented
- Initial experiments and evaluation conducted
